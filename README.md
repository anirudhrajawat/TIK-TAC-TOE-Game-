# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.  
This project is beginner-friendly and demonstrates the basics of DOM manipulation, game logic, and responsive UI design.

---

## 🚀 Features

- ✅ Two-player gameplay
- ✅ Interactive UI
- ✅ Winner detection logic
- ✅ Draw match detection
- ✅ Restart / Reset game option
- ✅ Responsive design for mobile and desktop

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the game
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Game functionality and logic
